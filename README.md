## What is Giganoto (Vision)?

- A motive:
  - You learn everything in life from the first principles.
  - You focus on becoming a great developer.
  - You love what You do and You do what You love!
  - You become more productive in life.
  - You become an open-minded individual who sees the truth of our broken and biased society.
  - You help yourself, and your loved ones and give back to the community!

- A complete open-source project where:
  - Front-end is open-source.
  - Design is open-source.
  - Decision-making is open-source/transparent.
  - Backend is open source.
  - Infrastructure is open source / run by anyone.
  - Content is open source.
  - Only User Data is private and encrypted for individual safety.

- A learning platform:
  - Focused on content that is considered high-quality and rarely free in computer science.
  - A practical no-nonsense content policy.
  - A learning-by-doing platform.
  - A gamified platform by the people for the people.

- A tool to find opportunities:
  - A group-based experience-driven opportunity curator.
  - A tool to find fellow learners and mentors.

- A community-driven platform:
  - Community should host the platform.
  - Community should take a vote about decisions.
  - Community should vote for quality.
  - Community should teach, learn, and grow!


## What is **NOT** Giganoto?

- A robot-like culture that focuses only on you getting a job and joining the rat race.
- Yet another website that teaches the same stuff as a million others! 
- A complex web of blogs and tutorials that only Google can find!


## Who can be the contributor?

- Anyone
  - who is eager and humble enough to interact with the community:
    - Ask for help when you need it.
    - Help others when you can.
  - consistent in problem-solving and learning.


## What's the plan?

####Phase 1: (Currently being coded)

  - Monolith front-end and back-end.
  - Hosted on community EC2 machines and Raspberry PIs.
  - learn about the backend in depth.
    - Run on our own Load balancer and HTTP server (Python).
    - Learn about HTTP Protocol and Network programming.
    - Learn about Distributed computers.
    - Learn about concurrency.
  - Host blogs by `content-creators` from the Giganoto community.
  - Voting (Only upvoting possible) from logged-in users.

#### Phase 2: (Coding starts from Nov, 15th)

  - Tech stack: Python, Javascript, HTML, CSS, React, Flask, Postgres, Redis.
  - Add learning by doing capability.
  - Introduce gamification in learning: detailed decisions yet to be taken.


## What kind of contributions are we looking for?
  - A Design choice for Giganoto UI. (colors, fonts, etc.)
  - Figma designs for a great UX.
  - Content (Great blogs on tech topics, it should have a "wow" factor).
  - Animations and Graphics that make learning fun and easy.
  - Frontend code if you can code in React/Next JS.
